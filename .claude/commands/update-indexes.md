# インデックス再生成 + 差分確認

entries/ の内容からインデックスファイル（会社別・ドメイン別・タグ別・日付別）と index.md を再生成し、変更内容を確認する。

## 引数

なし

## 実行手順

### ステップ1: インデックス生成

```bash
.venv/bin/python scripts/generate_index.py
```

### ステップ2: 差分確認

```bash
git diff indexes/ index.md
```

### ステップ3: サマリ表示

変更があった場合、以下を表示:
- 変更されたファイル一覧
- 追加・削除されたエントリ数
- 主な変更内容の要約

変更がない場合は「インデックスは最新です」と表示。
